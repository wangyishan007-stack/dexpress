### 核心流程变化

### GeckoTerminal 字段映射
| Pool 字段 | GeckoTerminal 字段 |
|---|---|
| price_usd | base_token_price_usd |
| liquidity_usd | reserve_in_usd |
| volume_* | volume_usd.* |
| txns_* | transactions.*.buys + sells |
| makers_* | transactions.*.buyers + sellers |
| change_* | price_change_percentage.* |
